## Overview

Bullnabi 팀의 Autonomous Landing Lead로서 VTOL UAV의 정밀 자율 착륙 시스템을 개발했습니다. AprilTag 비전과 센서 퓨전을 활용하여 선박 위 자율 착륙까지 기술을 확장했습니다.

## Key Contributions

- **Precision Landing**: AprilTag vision과 sensor fusion 기반 정밀 착륙 시스템 개발
- **Bézier Trajectory**: Bézier curve 기반 trajectory generation으로 smooth descent profile 구현
- **SITL/HITL Validation**: PX4/Gazebo 시뮬레이션 검증 후 실제 야외 비행 시험 성공
- **Technology Transfer**: 해상 선박 환경 자율 착륙 기술을 산업 파트너에 이전

## Awards
